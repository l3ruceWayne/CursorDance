# 更新日志

本文档记录 "CursorDance" 扩展的所有重要更新。

## [Unreleased]

### 变更

- macOS：打开文件/项目改为使用 URL Scheme（协议链接），支持 VS Code Insiders
- macOS：打开当前文件时先打开项目，再打开文件并定位到光标位置

## [1.0.0]

### 新增功能

- 快速在另一个编辑器中打开当前文件
- 自动定位到 VS Code 相同的光标位置
- 右键菜单直接打开文件到另一个编辑器
- 一键打开整个项目到另一个编辑器

## [1.0.1]

### Bug 修复

- 修复 macOS 上可能无法打开目标编辑器的问题

## [1.0.2]

### 代码优化

- 移除多余的快捷键配置

## [1.0.3]

### 代码优化

- 增加对 EPIPE 错误的捕获

## [1.1.0]

### 重大变更

- 移除旧版编辑器跳转逻辑与配置
- 支持 Cursor 与 VS Code 之间相互跳转（同文件、同位置）

## [1.1.1]

### 修复

- macOS 上通过 `open -a` 可能无法准确定位到指定行的问题，改为优先使用应用内置 CLI 执行 `--goto`

## [1.2.0]

### 清理

- 配置项前缀统一为 `switch2cursor.*`
- 命令 ID 前缀统一为 `Switch2Cursor.*`
